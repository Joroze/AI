AI Programming

This project creates and displays a user-defined dynamic 2D array grid where the user can define the grid size and percent of randomly generated obstacles within it.

The user is then be given a choice of algorithms that traverse the grid's user-defined start and end positions.

The webpage displays the process step by step, and then highlights the shortest possible route indicated.
